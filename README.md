# Grzybodziad 🍄

2D resource-collecting game built with Phaser 3. Collect mushrooms, berries, and flowers, then spend your points on upgrades!

**Play:** https://rogatty.github.io/grzybodziad/

## Development

```bash
npm install
npm run dev    # http://localhost:8080
```

Built with Phaser 3 + TypeScript + Vite.
